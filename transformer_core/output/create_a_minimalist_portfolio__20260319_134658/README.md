# Minimalist Portfolio - Clean & Accessible Professional Showcase

A modern, minimalist portfolio website designed for professionals seeking to showcase their work with elegance and accessibility. This single-page application features clean design principles, smooth interactions, and comprehensive accessibility features to create an optimal viewing experience for potential employers, clients, and recruiters.

## ✨ Features

### Core Features
- **Ultra-clean hero section** with prominent name display and brief introduction
- **Single column layout** with generous whitespace for optimal readability
- **Project showcase** with hover-triggered detail panels for interactive exploration
- **Concise about section** with professional biography content
- **Contact information** with click-to-copy functionality for easy contact
- **Smooth page transitions** for seamless navigation experience
- **Mobile-responsive typography** that scales appropriately across all devices
- **Reduced motion accessibility option** for users with motion sensitivity

### Additional Features
- Performance optimization for fast loading times
- SEO optimization for better search engine visibility
- Light/dark mode toggle for personalized viewing
- Analytics integration for visitor tracking
- Custom 404 error page for better user experience

## 🛠️ Technologies Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Modern styling with flexbox/grid and custom properties
- **JavaScript** - Interactive functionality and dynamic content
- **No external frameworks** - Pure vanilla implementation for optimal performance

## 🚀 Setup Instructions

Getting started with this portfolio is straightforward:

1. **Download** the project files to your local machine
2. **Navigate** to the project directory
3. **Open** `index.html` in your web browser
4. **No build process or dependencies required** - it works out of the box!

For development:
- Use any code editor (VS Code, Sublime Text, etc.)
- Live Server extension recommended for local development
- Ensure JavaScript is enabled in your browser

## 📖 Usage Guide

### Homepage (index.html)
The main landing page features a clean hero section with your name and introduction, followed by project showcases and about section.

### Project Detail (project-detail.html)
Dedicated page for individual project presentations with detailed descriptions and media.

### About Page (about.html)
Extended biography and professional background information.

### Contact Page (contact.html)
Contact form and information with click-to-copy functionality for easy contact.

### Navigation
- Smooth scrolling between sections
- Mobile-responsive navigation menu
- Keyboard navigation support

### Accessibility Features
- **Reduced Motion Toggle**: Click the accessibility icon to disable animations
- **Theme Toggle**: Switch between light and dark modes
- **Click-to-Copy**: Simply click on contact information to copy to clipboard
