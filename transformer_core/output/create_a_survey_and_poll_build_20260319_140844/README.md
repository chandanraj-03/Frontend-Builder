# SurveyBuilder: Create and Manage Surveys with Ease

## Project Description
SurveyBuilder is a web application designed to help create, manage, and analyze surveys with various question types, features, and analytics. This project aims to provide a user-friendly interface for survey administrators and respondents to collect valuable feedback from their target audience.

## Core Features

### Must-Have Features
- **Drag-and-drop Question Builder Interface**: Easily build your survey by dragging and dropping questions into the builder.
- **Question Types**: Support multiple question types, including:
  - Multiple Choice
  - Checkbox
  - Short Text
  - Rating Scale
  - Dropdown
- **Required / Optional Field Toggle per Question**: Customize your survey by marking specific fields as required or optional.
- **Shareable Survey Link Generation**: Share your survey with respondents via a shareable link.
- **Export Responses to CSV**: Download responses in CSV format for easy analysis.

## Setup Instructions
1. Clone or download this repository.
2. Open `index.html` in your favorite web browser to access SurveyBuilder.

## Usage Guide
### Creating a New Survey
1. Click on the "Survey Builder" page to start creating a new survey.
2. Drag and drop questions into the builder as needed.
3. Customize question types, toggle required/optional fields, and more from the drag-and-drop interface.

### Sharing Your Survey
1. Once you've built your survey, click on the "Share Survey" button to generate a shareable link.
2. Share this link with respondents via email, social media, or other channels.

### Analyzing Results
1. Click on the "Analytics" page to view real-time results.
2. Export responses to CSV for further analysis.
