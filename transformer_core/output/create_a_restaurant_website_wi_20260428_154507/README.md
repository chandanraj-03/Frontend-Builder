# DineBloom: Your Culinary Destination

DineBloom is a full-featured restaurant platform designed to streamline dining experiences through seamless reservations, ordering, and menu exploration. Our responsive design engages diners with intuitive features while maintaining accessibility for all users.

## Features

- **Real-time Reservation System**: Book tables instantly with automatic availability tracking
- **Dine-In/Takeaway Ordering**: Place orders for both service types with user-friendly interface
- **Smart Menu Navigation**: Filter dishes by dietary needs (vegan, gluten-free, allergies)
- **Interactive Dish Gallery**: High-quality image viewing with zoom functionality
- **User Reviews & Ratings**: Share feedback to help others make informed choices
- **Location Finder**: Integrated map showing restaurant address and directions
- **Instagram Integration**: Curated feed showcasing dishes, events, and behind-the-scenes
- **Event Calendar**: Stay updated with seasonal promotions and special occasions
- **Chef Profile**: Learn about our culinary team and philosophy
- **Parking Information**: Clear directions to on-site parking facilities
- **Mobile-First Design**: Fully responsive interface for smartphones and tablets
- **Newsletter Subscription**: Receive exclusive offers and menu updates
- **Digital Gift Cards**: Send personalized gift cards for easy gifting

## Technologies Used

- **HTML5** for semantic structure and accessibility
- **CSS3** for responsive layout and animations
- **JavaScript** for dynamic interactions (e.g., menu filters, reservation calendar)

## Setup Instructions

1. Download the project files
2. Open `index.html` in any modern browser
3. Explore all pages through navigation

## Usage Guide

| Page | Purpose | Key Features |
|------|---------|--------------|
| **index.html** | Home Page | Featured dishes, promotions, reservation widget |
| **shop.html** | Full Menu | Dietary filters, price sorting, dish descriptions |
| **product-detail.html** | Dish View | Nutritional info, photo gallery, add-to-cart |
| **cart.html** | Order Review | Items summary, quantity adjustment |
| **checkout.html** | Order Completion | Payment options, confirmation email |
| **login.html** | User Account | Order history, reservation management |
| **register.html** | Account Creation | Profile setup, subscription options |

The platform is fully responsive and optimized for mobile devices with touch-friendly controls.
