EcoStore

EcoStore is a comprehensive e-commerce platform designed for sustainable products. It features a responsive design, secure payment processing via Stripe, and an intuitive inventory management system. The platform was built with React for the frontend, Node.js and Express for the backend, and MongoDB for data storage.

Technologies Used:

  • React
  • Node.js
  • MongoDB
  • Stripe API
  • Redux for state management

HealthTrack

HealthTrack is a mobile application that helps users track their fitness goals and nutrition. It offers personalized workout plans based on user preferences and provides meal suggestions tailored to dietary needs. The app uses Firebase for authentication and data storage, and Redux for state management.

Technologies Used:

  • React Native
  • Firebase
  • Redux
  • Expo for cross-platform development

NewsSphere

NewsSphere is an AI-powered news aggregator that curates articles based on user interests and trending topics. It uses natural language processing to filter content and provide personalized news feeds. The backend is built with Express and MongoDB, and the frontend uses JavaScript with a responsive design.

Technologies Used:

  • JavaScript (ES6+)
  • Express
  • MongoDB
  • Natural Language Processing (NLP)
  • Web Scraping APIs
