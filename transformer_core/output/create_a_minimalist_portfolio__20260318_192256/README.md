# Minimalist Portfolio

A clean, accessible portfolio website designed for professionals to showcase their work with modern UI elements and responsive design.

## Key Features

- 🎯 **Ultra-clean hero section** with personalized introduction
- 🖼️ **Single-column layout** optimized for readability
- 📱 **Fully responsive design** for all device sizes
- 🔄 **Smooth transitions** between sections
- 🔍 **Interactive project showcase** with hover details
- ✉️ **Click-to-copy contact functionality**
- 🌐 **Reduced motion accessibility toggle**
- 📱 **Mobile-optimized typography scale**

## Technologies Used

- HTML5
- CSS3 (Flexbox/Grid)
- JavaScript (ES6)
- Pure CSS animations (no external libraries)

## Installation & Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/minimalist-portfolio.git
   cd minimalist-portfolio
   ```

2. Open the main file in your browser:
   ```bash
   open index.html  # macOS
   # or
   xdg-open index.html  # Linux
   ```

## Usage

1. **Hero Section**  
   - Personalized greeting and professional summary
   - Animated text reveal effect

2. **Projects Section**  
   - Hover to view project details
   - Click icons for GitHub/website links
   - Grid layout on desktop, single-column on mobile

3. **About Section**  
   - Concise professional bio
   - Skills visualization as progress bars

4. **Contact Section**  
   - Click-to-copy email button (no email clients needed)
   - Responsive contact form

5. **Accessibility Features**  
   - Reduced motion toggle (top-right corner)
   - High-contrast mode
   - Semantic HTML structure
