# TaskFlow - Streamlined Productivity Manager

## Description
TaskFlow is a modern, intuitive todo application designed to enhance personal productivity. With a clean interface and powerful features, it helps users organize tasks efficiently while providing visual feedback and persistent storage. Perfect for daily task management with seamless cross-device synchronization.

## Features
### Core Features
- ✅ Text input for adding new tasks
- ✅ Task completion with visual strikethrough
- ✅ Delete individual tasks
- ✅ Filter tabs (All, Active, Completed)
- ✅ Clear completed tasks button
- ✅ Real-time task count display
- ✅ Drag-and-drop task reordering
- ✅ Local storage persistence
- ✅ Empty state illustration

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)

## Setup Instructions
1. Download or clone the project files
2. Open `index.html` in any modern web browser
3. Start managing your tasks immediately - no installation required!

## Usage Guide
### Main Dashboard
The main interface provides a clean task management experience:
- **Add Task**: Type in the input field at the top and press Enter
- **Complete Task**: Click the checkbox next to any task
- **Delete Task**: Click the delete (X) button on the right side
- **Filter Tasks**: Use the All/Active/Completed tabs to view different task states
- **Reorder Tasks**: Drag and drop tasks to prioritize your workflow
- **Clear Completed**: Remove all completed tasks with one click

### Additional Pages
- **Task Detail View**: Click on any task to view/edit details
- **Settings Page**: Customize application preferences
- **Help & Tutorial**: Learn how to use TaskFlow effectively
- **Statistics & Analytics**: View your productivity trends and completion rates
