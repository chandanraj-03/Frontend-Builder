# CulinaryCraft: Your All-in-One Recipe Platform

## Description
CulinaryCraft is a comprehensive SaaS recipe platform designed for home cooks and food enthusiasts. This mobile-optimized application transforms kitchen routines through intelligent recipe search, meal planning, and collaborative sharing. With dietary filters, nutritional calculators, and kitchen-focused tools, it empowers users to cook with confidence while streamlining meal preparation and ingredient management.

## Features
### Core Features (Must-Have)
- **Dietary-Focused Search**: Filter recipes by vegan, gluten-free, keto, and other dietary needs
- **Step-by-Step Cooking**: Visual cues and timed instructions for each preparation stage
- **Smart Shopping List**: Generate and adjust ingredient lists with grocery app exports
- **Meal Planner Calendar**: Schedule meals, sync with existing calendars, and auto-populate shopping lists
- **Nutritional Calculator**: View calories, macros, and allergen information
- **User-Submitted Recipes**: Create and share recipes with structured submission fields
- **Rating & Reviews**: Star ratings, comments, and profile-based engagement
- **Video Tutorials**: Play/pause, speed controls, and step-skipping for cooking videos
- **Seasonal Collections**: Dynamic recipe collections based on weather and season
- **Printable Recipe Cards**: Generate customizable PDF/physical cards with toggleable dietary icons
- **Cooking Timer**: Persistent timer with step completion alerts
- **Unit Conversion Tools**: Context-aware measurement conversions (cups → mL)
- **Mobile-First Kitchen UI**: One-handed navigation, touch-friendly controls, and minimal text input
- **Social Sharing**: Pre-formatted snippets for Instagram/Pinterest sharing
- **Meal Prep Planning**: Batch-cook scheduling and ingredient prep checklists

## Technologies Used
- HTML5 (Semantic structure)
- CSS3 (Responsive layout with Flexbox/Grid)
- JavaScript (Client-side functionality)
- Modern CSS Features (CSS Variables, Flexbox, Grid)
- No backend required (Pure frontend implementation)

## Setup & Installation
1. Download the project files
2. Open `index.html` in any modern web browser (Chrome, Firefox, Edge)
3. Navigate through the application using the intuitive menu system
4. All features require no installation - fully functional in-browser

## Usage Guide
![CulinaryCraft Home Page](https://via.placeholder.com/800x400?text=Home+Page+Preview)
*Home page showcasing trending recipes and seasonal collections*

1. **Navigation**: Tap the hamburger menu (☰) to access all features
2. **Search**: Use filters for dietary needs (vegan/keto/glycemic index) and cooking time
3. **Recipe Detail**: 
   - View step-by-step instructions with timers
   - Expand shopping list with quantity adjustment
   - Toggle between video and text instructions
4. **Meal Planner**: 
   - Select dates and assign meals to your calendar
   - View auto-generated shopping list
   - Set meal prep reminders
5. **Community**: 
   - Share recipes via Instagram/Pinterest
   - Leave reviews with photo attachments
   - Follow other users' recipe collections
