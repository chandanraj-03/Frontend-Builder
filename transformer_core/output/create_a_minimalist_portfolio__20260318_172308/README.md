# PortfolioFlow - Minimalist Portfolio Website

## 📋 Description
PortfolioFlow is a clean, single-column portfolio website designed for professionals seeking to showcase their work with maximum clarity and minimal distraction. Featuring a mobile-responsive layout with generous whitespace, this portfolio emphasizes content while maintaining elegant design principles and accessibility standards.

## ✨ Features

### Core Features
- **Ultra-clean hero section** with prominent name display and introduction
- **Single-column layout** with generous whitespace for optimal readability
- **Project showcase** with interactive cards revealing details on hover
- **Concise about section** with professional biography
- **Contact information** with click-to-copy functionality for email and phone
- **Mobile-responsive typography** scale for consistent viewing across devices
- **Navigation system** for single-page portfolio sections
- **Content management capability** for easy updates to projects and bio

### Enhanced Features
- Smooth page transitions between sections
- Reduced motion accessibility option for users with motion sensitivities

## 🛠️ Technologies Used
- **HTML5** - Semantic markup structure
- **CSS3** - Responsive styling and animations
- **JavaScript (Vanilla)** - Interactive functionality

## 🚀 Setup Instructions

### Quick Start
1. Download or clone the project files
2. Navigate to the project directory
3. Open `index.html` in any modern web browser

### No additional dependencies or installation required!

## 📖 Usage Guide

### Portfolio Sections

#### Hero Section
- **Location**: Top of the page
- **Content**: Your name and introductory tagline
- **Visual**: Clean, centered text with ample whitespace

#### Project Showcase
- **Layout**: Single-column display of project cards
- **Interaction**: Hover over any project card to reveal additional details
- **Details Displayed**: Project description, technologies used, and links

#### About Section
- **Purpose**: Professional biography and background
- **Design**: Concise text in a readable format
- **Customization**: Easily update your bio information

#### Contact Section
- **Features**: Email, phone, and social media links
- **Functionality**: Click on email or phone to automatically copy to clipboard
- **Feedback**: Visual confirmation when information is copied

### Navigation
- Use the navigation menu to jump between sections
- Smooth scrolling transitions between content areas

### Accessibility Features
- **Reduced Motion**: Toggle available in settings for users who prefer reduced animations
- **Keyboard Navigation**: Full keyboard accessibility support
- **Screen Reader Optimized**: Semantic HTML structure for assistive technologies
